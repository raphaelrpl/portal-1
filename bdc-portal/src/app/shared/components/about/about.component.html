Brazil Data Cube

Brazil Data Cube is a project that is being developed by INPE (National Institute for Space Research), Brazil, that aims to create multidimensional cubes of analysis-ready data from medium-resolution earth observation satellite images for all Brazilian territory and to generate land use and cover information from these data cubes using machine learning and satellite image time series analysis.

The Brazil Data Cube is a subproject of the project “Environmental Monitoring of Brazilian Biomes“, funded by the The Amazon Fund in BNDES financial collaboration (Contrat nº 17.2.0536.1).
